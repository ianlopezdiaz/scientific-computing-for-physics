# Scientific Computing for Physics

> **Language:** 🇺🇸 English | 🇧🇷 [Português](https://github.com/ianlopezdiaz/scientific-computing-for-physics-pt)

A modern introduction to scientific computing for physics using Python.

This repository brings together educational materials, interactive notebooks, and computational examples designed to teach physics through scientific programming, data analysis, visualization, and numerical methods.

The goal is not simply to teach Python or implement algorithms, but to demonstrate how computational tools have become an essential part of modern scientific practice.

This project is being developed as a long-term educational resource and is intended to grow into a comprehensive companion for undergraduate Physics and Engineering courses.

---

## Objectives

The main goals of this project are to:

* Introduce the fundamentals of scientific computing using Python.
* Teach programming through the solution of physics problems.
* Develop skills in scientific data analysis and visualization.
* Introduce numerical methods gradually and in context.
* Complement traditional physics education with computational experiments.
* Provide open educational material for undergraduate courses.
* Build a free, continuously evolving learning resource.

---

## Project Philosophy

Scientific computing has become an indispensable tool for physicists, engineers, and researchers.

Throughout their academic careers, students encounter a wide range of computational tasks, including:

* analyzing experimental data;
* performing numerical simulations;
* creating scientific visualizations;
* solving differential equations;
* fitting mathematical models;
* applying statistical methods;
* producing publication-quality figures for reports and scientific papers.

This project aims to integrate all of these topics into a single learning pathway.

Rather than teaching programming as an isolated subject, Python is introduced as another mathematical and scientific tool for understanding physical phenomena, building computational models, and analyzing results.

## Project Organization

The repository is organized into four major parts.

### Part I — Scientific Computing

An introduction to the core tools used in scientific computing with Python.

Topics include:

* Python
* Jupyter Notebooks
* NumPy
* Pandas
* Matplotlib
* SciPy
* Scientific project organization

---

### Part II — Physics and Data Science

This section introduces the fundamental concepts of scientific data analysis with a focus on the needs of Physics and Engineering students.

Topics include:

* experimental data analysis;
* scientific visualization;
* descriptive statistics;
* measurement uncertainties;
* least-squares fitting;
* linear and nonlinear regression;
* communicating scientific results.

---

### Part III — Physics

The largest part of the repository is dedicated to the traditional undergraduate physics curriculum.

The content roughly follows the sequence of a standard university program:

* Physics I — Classical Mechanics
* Physics II — Waves, Fluids, and Thermodynamics
* Physics III — Electricity and Magnetism
* Physics IV — Optics and Modern Physics

Each notebook combines theoretical background, computational implementation, visualizations, simulations, and exercises.

Whenever appropriate, numerical methods are introduced naturally as practical tools for solving physics problems.

---

### Part IV — Numerical Methods

The numerical algorithms used throughout the repository are gathered in this section, where they are studied in greater depth.

Planned topics include:

* numerical differentiation;
* numerical integration;
* root finding;
* interpolation;
* linear systems;
* ordinary differential equations;
* partial differential equations;
* Monte Carlo methods;
* optimization.

This organization allows the physics notebooks to remain focused on physical concepts while providing a dedicated reference for the mathematical and computational foundations of the algorithms.

* undergraduate Physics students;

No previous programming experience is required, although a basic background in mathematics and physics is recommended.
